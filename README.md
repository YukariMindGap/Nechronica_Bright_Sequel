# Nechronica: Bright Sequel
New and reworked version of the fork for Malkeus's Compatible Nechronica.

Basically it's a mod about zombie girls (called Dolls) in a post-apocalyptic world fighting against zombies employed by the Necromancer (the DM). 
I'm not sure if i should somehow integrate this with pre-existing lore.

The difference between the old fork and this one is that this mod will mainly be compatible with Cataclysm: Bright Nights, as that fork is what i'm focusing on, and this one fixes all the problems with the old mod.
